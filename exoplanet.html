Поиск обитаемых экзопланет

Это интеллектуальная система для анализа и выявления потенциально обитаемых экзопланет. Она использует методы машинного обучения для оценки планет на основе их физических характеристик.

Описание проекта

Система использует передовые методы машинного обучения для обработки данных из авторитетных источников, таких как NASA Exoplanet Archive и Planetary Habitability Laboratory.

Основная функциональность включает оценку планет по их физическим характеристикам (радиус, температура, плотность), моделирование состава атмосферы, анализ водного потенциала и общей пригодности для жизни. Взаимодействие с данными осуществляется через API, предоставляя доступ к каталогам подтвержденных экзопланет и потенциально обитаемых миров.

Ключевые технологии

  • Python: Основной язык реализации.
  • Flask: Веб-фреймворк для создания API и интерфейса.
  • Scikit-learn: Для реализации моделей машинного обучения (KNN регрессор, Random Forest).
  • Pandas & NumPy: Для обработки и анализа данных.
